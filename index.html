Site Guide

What this site is for

This site is the PD Planner Tool for planning professional development across the OptimaAcademy Online network of schools.

It brings the yearlong PD schedule, build status, topic catalogue, recommendation pools, topic groups, links, notes, and planning evidence into one shared planning board.

Sign-in and shared access
  • Microsoft sign-in: users must sign in before the site can be used.
  • Shared data: changes save to Firestore and can update live for other signed-in users.
  • Sign out: the sign-out button appears in the top header after sign-in.
Top-right buttons
  • Group Key: opens the left reference panel showing topic ID prefixes and what each group means.
  • Site Guide: opens this right-side searchable guide while the rest of the site remains usable.
  • + New Topic: opens the topic editor to create a new PD topic.
  • Sign out: signs the current Microsoft user out of the planner.
Main navigation tabs
  • Schedule: the master calendar view for PD weeks, schools, session types, topics, and recommendations.
  • Build Tracker: a Kanban-style board for moving scheduled PD work through production stages.
  • Topic Catalogue: the searchable topic database with descriptions, outcomes, and evidence.
  • Recommendations: editable recommendation pools used by the schedule helper buttons.
Schedule tab
  • Week/date rows: each row represents a PD week or special schoolwide event.
  • School columns: show which school or teacher group has PD that week.
  • Session type labels: identify Live, Subject Live, Async, onboarding, showcase, conference, party, and closeout slots.
  • Topic tags: colored topic IDs assigned to a schedule slot. Click a topic tag to open its details.
  • Topic Pool: draggable topic tags that can be filtered by Live, Subject, Async, or All.
  • Drag and drop: drag a topic from the pool or from one slot into another schedule slot.
  • Remove X: the red X appears when hovering over a scheduled topic and removes that topic from that slot.
Recommendation helper buttons

The small recommendation button in a schedule slot opens suggested topics for that school, session family, quarter, subject, or audience.

Use it when a slot is empty or when stakeholders want to understand which PD options were planned for that type of session.

Build Tracker tab
  • Stage columns: Planning, In Build, Review, Ready, and Delivered.
  • Cards: each card represents a scheduled PD slot with assigned topics.
  • Drag cards: move cards between stage columns to update production status.
  • Card details: cards can show school, session type, week, topic, links, and notes.
  • Click a card: opens the schedule/session editor for that PD slot.
Topic Catalogue tab
  • ID: the short code used everywhere else in the planner.
  • Title: the full PD topic name.
  • Lean: marks the topic as Technical, Pedagogical, or Blended.
  • Description: explains what the PD session is about.
  • Outcomes: names what teachers should leave with.
  • Evidence: captures the research or rationale behind the topic.
  • Click topic IDs: opens the editable topic detail modal.
Recommendations tab

Recommendation pools are reusable sets of suggested topics. They support the schedule helper button and help explain why certain topics fit certain PD slots.

  • Pool label: names the recommendation context.
  • School/family/quarter fields: connect the pool to matching schedule slots.
  • Items/topic IDs: the topics suggested for that context.
  • Rationale: explains the planning reason behind the pool.
Group Key panel

The left Group Key panel explains topic ID prefixes, such as VR-T, AI-P, LOW, K1, G4, G5, and AS.

Each group shows its prefix, name, format type, topic count, and description. The Add Group button creates a new prefix group.

Topic and session editors
  • Topic editor: create or edit topic ID, title, lean, description, outcomes, and evidence.
  • Used by: shows where a topic appears in the current schedule.
  • Session editor: edit scheduled slot details, assigned topics, notes, and file links.
  • File links: attach URLs to planning docs, PD materials, recordings, or supporting resources.
Colors, badges, and labels
  • Blue/cyan topic tags: Live or technical-style PD topics.
  • Green topic tags: Subject or school-specific PD topics.
  • Purple topic tags: Async self-paced modules.
  • Lean badges: Technical, Pedagogical, and Blended describe the instructional focus of a topic.
  • Stage dots/status: show where a session is in the build process.
Export Current View

Export Current View creates a text export of the schedule view currently being shown, respecting active filters. It is useful for sharing with stakeholders or pasting into an LLM for analysis.

The export includes the currently visible schedule information and the descriptions of topics currently within that schedule view.

How to use this guide
  • Type in the search box to narrow the guide to matching headings and keywords.
  • Open or close headings to read only the section you need.
  • Leave the guide open while clicking tabs, opening topic details, or reviewing the schedule.
